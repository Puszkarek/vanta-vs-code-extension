# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-02-01

### Released
- **Void Reborn**: Initial release of the "Void" extension.
- a simple TypeScript playground inside your editor.

### Features
- **Realtime Execution**: Runs TypeScript code on-the-fly as you type.
- **Inline Logs**: Displays console output directly next to your code.
- **Commands**:
    - `Void: Enter the Void (Current File)`: Attach to active file.
    - `Void: Summon Fresh Void (New File)`: Create and run a new playground file.
- **Configuration**:
    - `void.debounce`: Control execution delay.
    - `void.tsconfigPath`: Explicitly set tsconfig location.
